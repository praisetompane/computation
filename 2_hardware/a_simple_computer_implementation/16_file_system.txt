def file system:

    formal (Linux System Administrators Guide): a datastructure and method
                that provides a way
                    to manage how
                        data is stored and retrieved from disk.
    plain english (Scott, 2009:115):

        a collection of programs
            provided by the Operating System
        that make the disk (heads, tracks, sectors and block of bytes)
            look like folders and files
                in a file cabinet.
